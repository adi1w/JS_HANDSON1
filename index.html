Q8.What is the difference between window vs document?

window

  1. Represents the global window in a web browser
  2. servers as the top- level object in the browser's window hierarchy.
  3. contains properties and Methods related to thye browser window, such as dimensions,locations,history,navigation.
  4. Provides access to other global objects, such as document , navigators, and localstorage.
  5. Acts as the execution context for global Javascript Code.
  6. Handles events related to the window, such as load,resize,and scroll
  7. Can be used to open new browser windows or tabs using methods like open() and close().

Document

  1. Represents the web page loaded within the browser window.
  2. Provides access to the elements and content within the web page.
  3. contains methods and properties to manipulate and interact with the document's structure, content and styling.
  4. Enables the selection of HTML elements using various methods like getElementById() , querySelector() , or getElementsByTagName()
  5. Handles events that occur within the document , such as click,submit and keydown.
  6. Allows modification of element properties, such as text content, HTML markup , attributes and styles
  7. Provides methods to create new element , modify the document structure ,or dynamically load external resources.
